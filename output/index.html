About JDay Dominican Republic

JDay Dominican Republic is a community driven conference organized by the Dominican Republic Java User Group (Aka JavaDominicano). It is designed to help developers to learn the newest technologies about Java, Web, Mobile, Cloud and DevOps. Besides, developers, tech companies, and startups can establish a good network among them and for them who want to enjoy the pristine beaches, cloud-forest treks and welcoming villages, this is a great opportunity to come to the Dominican Republic to learn about among Java topics and have fun sightseeing.
