Conference Info

JConf Dominicana is a Java technologies conference organized by the Dominican Republic Java Users Group (Aka JavaDominicano). It is designed to discuss the latest about the Java ecosystem, including alternatives JVM languages, Web and Mobile development, and newest Cloud tendencies.

Due to covid-19 pandemic the JConf Dominicana 2021 edition will be virtual.
