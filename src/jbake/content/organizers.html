Organizers

JConf Dominicana is a Java technologies conference organized by the Dominican Republic Java Users Group (Aka JavaDominicano).

Event Committee

The sessions and keynotes for JConf Dominicana 2022 are chosen by a committee made up of volunteers from Dominican Republic Java Users Group (Aka JavaDominicano).
